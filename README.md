- Index.html aka /home. This is the way to learn about me in a short time frame (click over the wavy line text to open a pop-up).
- Projects.html aka /projects. This is the table that stores my projects with clickable links. there is tags to sort by category as well.
- Research.tml aka /research. This shows my publications and current research focus.
